## Getting Started

1. Clone with submodule for the first time.
  - git clone --recursive https://github.com/chungchiehlun/dotfiles.git
2. Pull with submodule to update submodules.
  - git pull --recurse-submodules

```bash
$ source bootstrap.sh

// macOS: Terminal
$ source terminal.sh

// windows: Git-Bash
```
